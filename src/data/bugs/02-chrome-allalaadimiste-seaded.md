---
title: "Chrome seadetes peab muutma allalaadimiste seadeid"
problem: "Laienduse töötamise ajaks tuleb Chrome'i seadetes (Settings → Downloads) deaktiveerida 'Ask where to save each file before downloading' seade. Muidu tuleb iga pildi allalaadimisel asukoht nupulevajutusega kinnitada ja see nullib extensioni kasu automatiseerimise osas ära."
workaround: "Avage Chrome Settings → Downloads → lülitage välja 'Ask where to save each file before downloading'. Pärast laienduse kasutamist võite seade tagasi sisse lülitada."
status: "active"
severity: "high"
order: 1
