answer: "Pead olema oma lapse rühmapäeviku lehel. URL näeb välja umbes nii: eliis.eu/child/123456/diary. Mine 'Minu laps' → 'Rühmapäevik'."

Laiendus töötab ainult rühmapäeviku lehel, kust ta saab lapse info kätte.

**Kuidas sinna jõuda:**
1. Logi Eliisi sisse
2. Vali "Minu laps"
3. Vali "Rühmapäevik"
4. URL peaks olema kujul: `eliis.eu/child/[number]/diary`
